# Rusteroids

This is a clone of the classic arcade game Asteroids in
[Rust](http://rust-lang.org). It's purpose is to help me learn the language.

# Build

You must have SDL2 installed. (`brew install sdl2` with Homebrew).

```sh
make exe
```
